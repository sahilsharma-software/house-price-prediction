# 🏠 House Price Prediction App

A Machine Learning web application that predicts house prices based on user inputs like number of bedrooms, bathrooms, and living area.

The app is built using **Python, Scikit-learn, and Streamlit** and deployed online so anyone can use it.

## 📌 Features

* Predict house prices instantly
* Simple and interactive UI
* Built using Streamlit
* Machine Learning model using Random Forest
* Deployed online for public access

## ⚙️ Installation

Clone the repository:

git clone https://github.com/sahilsharma-software/house-price-prediction.git

Go to the project folder:

cd house-price-prediction

Install dependencies:

pip install -r requirements.txt

Run the Streamlit app:

streamlit run app.py

---

## 📊 Model Information

The project uses a **Random Forest Regressor** model trained on housing data.

### Input Features

* Bedrooms
* Bathrooms
* Sqft Living Area

### Output

* Predicted House Price
